# Quinzenas — PWA

Controle financeiro por quinzena com sincronização na nuvem.

## Como publicar no GitHub Pages

### 1. Crie um repositório no GitHub
- Acesse github.com e clique em **"New repository"**
- Nome: `quinzenas` (ou qualquer nome)
- Deixe como **Public**
- Clique em **"Create repository"**

### 2. Faça upload dos arquivos
Na página do repositório criado, clique em **"uploading an existing file"** e arraste todos estes arquivos:
```
index.html
manifest.json
sw.js
icons/
  icon-192.png
  icon-512.png
```

### 3. Ative o GitHub Pages
- Vá em **Settings** → **Pages**
- Em "Source", selecione **"Deploy from a branch"**
- Branch: **main**, pasta: **/ (root)**
- Clique em **Save**

### 4. Seu app estará disponível em:
```
https://M5RDINH4.github.io/quinzenas/
```
(substitua `quinzenas` pelo nome do repositório que você criou)

## Instalar no celular

### Android (Chrome ou Zen Browser)
1. Abra o link do app no navegador
2. Toque nos 3 pontos → **"Adicionar à tela inicial"** ou **"Instalar app"**
3. Pronto — aparece o ícone na tela igual a um app

### iPhone (Safari)
1. Abra o link no **Safari** (obrigatório no iOS)
2. Toque no ícone de compartilhar (quadrado com seta)
3. **"Adicionar à Tela de Início"**
4. Pronto

## Usar em vários dispositivos
Abra o mesmo link em qualquer dispositivo, entre com o mesmo **usuário e senha** — os dados sincronizam automaticamente.
